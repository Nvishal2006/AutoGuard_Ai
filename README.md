# Agentic AI Predictive Maintenance Ecosystem

This project is a full-stack prototype of an Agentic AI system for Automotive OEMs. It uses a Master Agent to orchestrate multiple worker agents for analysis, diagnosis, engagement, scheduling, and insights.

## System Architecture

- **Backend**: Python FastAPI
  - **Agents**: Analysis, Diagnosis, Engagement, Scheduling, Feedback, Insights, Master.
  - **ML Model**: RandomForest Classifier (Scikit-learn).
  - **Data**: Synthetic telematics data.
- **Frontend**: Next.js (React)
  - **Dashboard**: Real-time status, Scheduling, OEM Insights.
  - **Voice Bot**: Interactive assistant.

## Prerequisites

- Python 3.9+
- Node.js 18+

## Installation & Setup

### 1. Backend Setup

```bash
cd backend
pip install -r requirements.txt
# Train the model and generate sample data
python ml/train_model.py
python generate_samples.py
```

### 2. Frontend Setup

```bash
cd frontend
npm install
```

## Running the Prototype

### 1. Start the Backend

Open a terminal and run:

```bash
cd backend
uvicorn main:app --reload
```
The API will run at `http://localhost:8000`.

### 2. Start the Frontend

Open a new terminal and run:

```bash
cd frontend
npm run dev
```
The dashboard will be available at `http://localhost:3000`.

## Demo Flow

1. **Real-Time Status**: View the main dashboard. The charts update in real-time via WebSocket.
2. **Predict Failure**: Click "Run AI Prediction". The agents will analyze the data, calculate risk, and the Voice Bot will pop up with a message.
3. **Voice Bot**: Interact with the bot. Type "Book it" to confirm an appointment.
4. **Scheduling**: Go to the "Service Scheduling" page to see the auto-booked appointment.
5. **Insights**: Go to "OEM Insights" to see RCA and CAPA recommendations generated by the Insights Agent.
