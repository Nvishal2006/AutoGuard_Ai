# Demo Script: Agentic AI Predictive Maintenance

**Duration**: 3 Minutes

## Scene 1: Introduction (0:00 - 0:30)
- **Visual**: Show the "Real-Time Status" Dashboard.
- **Narrator**: "Welcome to the Agentic AI Predictive Maintenance Ecosystem. Here we see real-time telematics data streaming from a connected vehicle."
- **Action**: Point out the live charts for Brake Wear, Engine Temp, and Vibration.

## Scene 2: Failure Prediction & Agent Orchestration (0:30 - 1:15)
- **Narrator**: "Let's simulate a failure scenario. The Analysis Agent detects an anomaly in the brake wear patterns."
- **Action**: Click the **"Run AI Prediction"** button.
- **Visual**: The "Alert Panel" appears showing a "High Risk" or "Critical" alert. The Voice Bot opens automatically.
- **Narrator**: "The Diagnosis Agent calculates a SUC score (Severity, Urgency, Confidence) of 0.82, triggering a Critical Alert. The Engagement Agent immediately notifies the user."

## Scene 3: Voice Bot Interaction (1:15 - 2:00)
- **Visual**: Voice Bot chat window.
- **Narrator**: "The user can interact naturally. The bot suggests a service slot based on the Scheduling Agent's optimization."
- **Action**: Type **"Book it"** or **"Confirm"** in the chat.
- **Visual**: Bot confirms the booking.
- **Narrator**: "The Scheduling Agent has now auto-booked the appointment at the nearest center with the least load."

## Scene 4: Scheduling & Insights (2:00 - 3:00)
- **Action**: Navigate to **"Service Scheduling"**.
- **Visual**: Show the new appointment in the list and the Service Center Load chart.
- **Action**: Navigate to **"OEM Insights"**.
- **Visual**: Show the "Top Recurring Failures" chart and "CAPA Recommendations".
- **Narrator**: "Finally, the Insights Agent performs Root Cause Analysis. It identifies a batch issue with brake pads and suggests a specific CAPA action, closing the loop for the OEM."
